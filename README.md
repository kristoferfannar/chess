# Chess

This project aims at building a tiny chess platform. 
This includes creating an interface for playing chess, but also a platform for developing chess engines.

The project is currently deployed on [kristoferfannar.com/chess](https://kristoferfannar.com/chess).

## Building

### I don't want to think
```bash
docker-compose up --build
```

### I want hot reload

```bash
bun run dev
```

Then open [http://localhost:3000](http://localhost:3000).
